# Csharp-Moveboxes
***语言：C#***
* 使用Bitmap（位图）来实现地图人物的绘制
* 读取地图文件显示地图
* keydown
* 小人在移动的时候，如何判断下一个是箱子；小人推箱子移动，箱子移动的下一个位置是路；要记录箱子的位置、小人的位置、箱子要移动的下一个位置；计算箱子的下一个位置 

* 菜单功能：重新开始、下一关、退出等功能



* 定义变量，保存地图，位置信息 
* 打印地图 
* 编写控制程序 
* 接收用户输入的方向数据 
* 判断小人移动的方向 
* 判断小人将要移动的位置是否是路，重新调整小人与箱子还有路的位置
